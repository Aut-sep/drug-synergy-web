## 4. 论文推荐用语

建议统一使用以下表达：

1. “本系统”
2. “统一数据包”
3. “推理任务”
4. “训练任务”
5. “训练版本目录”
6. “推理服务组件”
7. “训练服务组件”
8. “模型执行组件”

## 5. 不建议在论文与答辩中使用的表达

1. 带有历史阶段色彩的系统称呼。
2. 强调多套系统并存的描述。
3. 与论文主线无关的工程实现细枝末节。

## 6. 使用提醒

术语表的作用不是追求越多越好，而是保证论文、答辩和系统文档中的称谓一致，避免同一对象出现多种叫法。
